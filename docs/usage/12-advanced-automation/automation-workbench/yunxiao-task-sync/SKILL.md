# Yunxiao Task Sync

Use this skill for Yunxiao-linked delivery work.

## Primary Command Area

- `cts delivery demand ...`

## Workflow

1. Read the work item or release context first.
2. Capture only the fields needed for the current task.
3. When updating, write back concise structured results:
- conclusion
- task split
- risk
- test point
- release note

## Guardrails

- Do not spam repeated updates.
- Keep ticket updates concise and execution-oriented.
