# Travel Web Research

Use this skill for web-supported travel context.

## Primary Command Area

- `cts travel search ...`

## Workflow

1. Search only for the facts that the route plan still lacks.
2. Prefer official or practical sources for hours, closures, and local rules.
3. Summarize the travel impact of each finding.

## Guardrails

- Search is supplementary, not the transport source of truth.
- Point out when information may be stale or should be rechecked.
