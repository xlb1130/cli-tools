# City Route Planning

Use this skill for map-centric travel planning.

## Primary Command Area

- `cts travel map ...`

## Workflow

1. Confirm start point, destination, date, and rough departure time.
2. Compare likely route options and travel durations.
3. Call out transfer burden, estimated time, and convenience tradeoffs.
4. Summarize the most practical option.

## Guardrails

- Prefer real route and travel-time evidence over intuition.
- If exact addresses are unclear, say that location precision may affect timing.
